# StrixReady

## Getting started

Requirements:

- Node.js 18+
- npm

Install dependencies and run locally:

```sh
npm install
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

Run tests:

```sh
npm run test
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
